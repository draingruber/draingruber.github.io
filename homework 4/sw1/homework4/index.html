My PWA

A PWA is a Progressive Web Application. It is a new class of websites that allow for native app experiences in just a browser. A PWA is distinguished by its use of push notifications, cache, and secure connection. The PWA must have usability offline as well. The tag line is that a PWA provides a rich web-based experience.

A PWA must be secure and is only available over HTTPS. It must have a manifest file that determines the experience and behavior of the PWA and be installable from the menu. It must have offline capabilities and be responsive.
